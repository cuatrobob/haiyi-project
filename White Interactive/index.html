This is a simple visualization of an application to the University of Minnesota.

Part of the admission process was done with the help of an SVM machine learning algorithm that was trained and tested with previous years' application data to accurately match our admission committee's decision process. This means the algorithm calculates numerical scores for each input, weights them, and sums the result to determine a final score for each user, which is then compared with an acceptance threshold. If the score is higher, the person is accepted. You can tweak the values below to find out the exact contribution of each score, and see the total impact on the decision. The contribution of each value is color-coded in the bar.
